# CMDB 资产管理系统



<br>
随着互联网的进步与发展，企业的IT环境越来越复杂。数量庞大、品种繁多的IT设备很难被有效的管理，更不用说管理以这些IT设备为基础的各种IT服务。CMDB正是来解决这些问题的，同时为运维自动化，监控报警等提供支持。

开发环境：Windows10 + Python3.5 + PyCharm5<br>
系统架构：Python3.5 + sqlite3 + Django1.10 + AdminLTE
