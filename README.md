# CMDB 资产管理系统



<br>
随着互联网的进步与发展，企业的IT环境越来越复杂，数量庞大、品类繁多。如何有效地管理IT设备，降低成本，并为业提供高效的IT服务，成为很多企业面临的问题。CMDB正是来解决这些问题的，同时为运维自动化，监控报警等提供支持。

开发环境：Windows10 + Python3.5 + PyCharm5<br>
项目架构：Python3.5 + sqlite3 + Django2.1 + AdminLTE
